# RutinKu

**Bangun rutinitas harianmu, satu centang setiap hari.**

Aplikasi habit tracker sederhana untuk membantu pengguna membangun rutinitas harian secara konsisten. Dibangun dengan React Native (Expo) dan disimpan secara lokal menggunakan AsyncStorage.

![Platform](https://img.shields.io/badge/platform-Android-3DDC84?style=flat-square&logo=android)
![React Native](https://img.shields.io/badge/React%20Native-Expo-000020?style=flat-square&logo=expo)
![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)

## Fitur Utama

- ✅ **Tambah & kelola rutinitas harian** — buat rutinitas baru dengan nama dan ikon pilihan sesuai kebiasaan yang ingin dibangun
- 🔥 **Tracking harian dengan streak counter** — tandai selesai setiap hari dan lihat berapa hari beruntun kamu konsisten
- 💾 **Penyimpanan otomatis di perangkat** — data tersimpan lokal via AsyncStorage, tetap ada walau app ditutup
- 🎨 **Tampilan bersih & responsif** — UI modern yang nyaman dipakai sehari-hari di Android

## Cara Install

1. Download file APK melalui link di atas
2. Buka file APK yang sudah terdownload di HP Android
3. Izinkan instalasi dari sumber tidak dikenal jika diminta oleh sistem
4. Tunggu proses instalasi selesai, lalu buka aplikasi **RutinKu**

## Tech Stack

| Teknologi           | Kegunaan                                             |
| ------------------- | ---------------------------------------------------- |
| React Native (Expo) | Framework utama pengembangan aplikasi mobile         |
| Expo Router         | Navigasi antar layar berbasis file-based routing     |
| TypeScript          | Type safety untuk kode yang lebih aman & terstruktur |
| AsyncStorage        | Penyimpanan data lokal di perangkat                  |
| EAS Build           | Build & distribusi APK Android                       |

## Setup Development

Clone dan jalankan project ini secara lokal:

\`\`\`bash
git clone https://github.com/rdniell/Rutinku.git
cd Rutinku
npm install
npx expo start
\`\`\`

Scan QR code yang muncul dengan aplikasi **Expo Go** di HP Android/iOS untuk mencoba app dalam mode development.
