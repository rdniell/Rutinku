# RutinKu

Aplikasi habit tracker sederhana untuk membantu pengguna membangun rutinitas harian.

## Fitur Utama

✅ Tambah & kelola rutinitas harian dengan ikon pilihan
🔥 Tracking progress harian dengan penghitung streak (hari beruntun)
💾 Data tersimpan otomatis di perangkat (AsyncStorage)
🎨 Tampilan bersih dan responsif di Android

## Tech Stack

React Native + Expo (Expo Router)
TypeScript
AsyncStorage untuk penyimpanan lokal
EAS Build untuk distribusi APK



## Download APK

[Download RutinKu APK](https://expo.dev/accounts/niell77/projects/rutinku/builds/93df5a29-301b-437a-a31e-63a170fa5291)

## Cara Install

1. Download file APK dari link di atas
2. Buka file APK di HP Android
3. Izinkan instalasi dari sumber tidak dikenal jika diminta
4. Buka aplikasi RutinKu
